COMO FUNCIONA UNA WEB

1 PONEMOS LA URL
2 NAVEGADOR PIDE AL SO SI TIENE EN CACHE EL DNS RESUELTO
3 NAVEGADOR FORMA UNA HTTP REQUEST, CREA UN HEADER (GET /FOTOS HTTP HOST USER-AGENT: CHROME)
4 EL SERVIDOR RECIBE ESTE HTTPREQUEST POR EL PUERTO 80
5 EL SERVIDOR RESPONDE CON HTTP RESPONSE (INDICANDO 200 OK, ULTIMA FECHA DE MOD, TIPO, CONTENIDO HTML)
6 SI SE PIDE OTRO FORMATO SE REPITEN LOS PASOS
